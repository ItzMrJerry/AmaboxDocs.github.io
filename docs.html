Getting Started

Start by downloading Unity Hub

Unity Hub

After downloading unity hub go to installs and Click on "Install Editor" and install the 2022 LTS(Long Term Support) version.

After Installing the editor go to projects and Click on "New Project".

After Clicking on New project, Make sure your using the correct editor version which can be changed at the top.
Then select from the templates Universal 3D(URP) and give your project a Name and Click "Create Project".

Small Description

RealVirtual & Amabox Installation

How to Install RealVirtual and Amabox in unity

Download this unity package

When you project has opened Go to Assets, Import Package and then Custom Package select the package you just downloaded.

After the package has Loaded click on "Import" and let the package install this can take a while.

Center Pivot To Mesh

Description of the Center Pivot To Mesh tool.

Select an object with a Meshrenderer component.
It checks if the pivot point is centered with the mesh center.

Debug Mesh Info

Description of the Debug Mesh Info tool. Shortcut: Alt+D

Select an object.
It checks al child objects if it has a MeshFilter and debugs the combined triangle count and other usefull information.

Material Setter

Description of the Material Setter tool. Shortcut: Alt+S

A tool that is made to easily set the color of a object.
it has multiple material types to choose from and can add new material types and materials in the resource folder

Part Connector

Description of the Part Connector tool. Shortcut: Alt+L

A tool To easily align Amabox structures with eachother.
select a part link for part 1 and for part 2. then Click on "Link Parts" it moves part 1 to part 2.

SubMesh Extractor

Description of the SubMesh Extractor tool.

Some Meshes have submeshes that can not be moved seperatly they are connected to eachother.
this tool breaks those connections and makes them seperate meshes.
